# Research Plan: Hermes Skill vs Claude Skill 对比 & CLAUDE.md 对标 Hermes 文件

## 核心研究目标

1. **Hermes 是什么**：Hermes 系统的定义、架构、设计理念
2. **Hermes Skill 机制**：Hermes 中 skill 的定义方式、工作原理、配置方法
3. **Claude Skill 机制**：Claude Code 中 skill 的定义方式、工作原理、配置方法
4. **对比分析**：两者在技能系统设计上的异同
5. **文件对标**：CLAUDE.md 在 Hermes 体系中对应哪个文件

## 研究策略：Breadth-first query

拆分为 4 个独立子任务并行研究：

### Subagent 1: Hermes 系统概述与 Skill 机制
- 搜索 Hermes AI 编程助手/框架的相关信息
- 了解 Hermes skill 的定义方式、文件结构、工作原理
- 关键词：Hermes AI coding agent, Hermes skill system, Hermes agent framework

### Subagent 2: Claude Code Skill 机制
- 搜索 Claude Code skill 的定义方式、文件结构、工作原理
- 了解 CLAUDE.md 的作用和定位
- 关键词：Claude Code skill system, CLAUDE.md, Claude Code custom slash commands

### Subagent 3: 微信公众号文章检索
- 使用 wechat-article-search skill 检索中文社区关于 Hermes 和 Claude skill 的讨论
- 关键词：Hermes AI编程, Claude Code skill, AI编程助手对比

### Subagent 4: 技术社区对比分析
- 搜索 GitHub、技术博客等平台对两者的对比讨论
- 了解社区对两者的评价和实际使用体验
- 关键词：Hermes vs Claude Code, AI coding assistant comparison 2025 2026

- 时间范围：2024-2026
- 关键词组合：Hermes skill, Claude skill, AI coding agent, CLAUDE.md, Hermes config
