# Visword Prototype v2

基于 `research_report_visword_infoflow_v2.md` 生成的产品原型。
聚"AI 热点信息流生成 + 二次编辑"方向。

## 使用方式

```bash
python3 -m http.server 8080
# 或直接双击 index.html 在浏览器中打开
```

## 与 v1 的区别

| 维度 | v1 | v2 |
|---|---|---|
| 产品方向 | 三合一（思维导图+信息卡+词云）| 热点信息流+编辑 |
| 用户输入 | 粘贴文字/URL | 浏览 AI 策展的信息流 |
| 核心交互 | 输入→生成→三标签切换 | 浏览→选中卡片→编辑→分享 |
| 编辑体验 | 有限（配色/尺寸切换）| 核心差异点（widget式直接编辑）|
| 设计语言 | 浅色主题 | 深色主题（资讯App风格）|
| 移动端 | 响应式适配 | Mobile-first，含手机框演示 |

## 原型覆盖的页面

1. **首页** — 产品价值主张 + 手机框动态预览 + 话题分类 + 定价
2. **信息流浏览** — 卡片网格 + 话题筛选 + 模拟实时热点数据
3. **卡片详情+编辑器** — 预览/编辑双栏布局，widget式编辑器（标题/摘要/要点/标签/主题色/尺寸）
4. **分享模态** — 复制链接 + Twitter/LinkedIn/Instagram 一键分享
5. **定价页**

## 交互特性

- 话题筛选切换（All/AI/Tech/Startups/Science/Business）
- 点击卡片进入详情+编辑器
- 编辑模式切换（✏️ Edit Card / ✓ Done）
- 双向同步：编辑器改动实时反映到预览区
- 添加/删除要点
- 5套主题色 + 4种社交尺寸
- 导出 + 分享
- 响应式适配
