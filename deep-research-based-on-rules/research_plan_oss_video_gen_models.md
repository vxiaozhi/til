# Research Plan: Open-Source AI Video Generation Models for E-Commerce SaaS (2026)

## Query Type: Depth-first + Breadth-first (hybrid)
- Multiple models to evaluate individually (breadth)
- Each requires deep technical analysis (depth)

## Research Objectives
1. Identify and evaluate ALL viable open-source AI video generation models
2. Assess each for e-commerce product-to-video-ad use case
3. Calculate GPU self-hosting costs vs API costs
4. Provide a ranked recommendation

## Subagent Plan

### Subagent 1: Tier 1 Production Models (WAN 2.5/2.6, LTX 2.3, Bernini)
- GitHub stats, license, GPU requirements, image-to-video quality
- Self-hosting cost calculations
- Search: WAN 2.5 video generation open source, LTX 2.3 Lightricks, Bernini ByteDance open source

### Subagent 2: Tier 1 Asian Models (SkyReels-V3, Hunyuan Video, MOVA, CogVideoX)
- GitHub stats, license, GPU requirements, image-to-video quality
- Search: SkyReels-V3 GitHub, HunyuanVideo open source, MOVA OpenMOSS, CogVideoX 2026

### Subagent 3: Tier 2 Models + GPU Economics (HappyHorse, Seedance, SVD, + all new 2026 models)
- Check if these are actually open-source or API-only
- Find any new 2026 open-source video models
- Search: HappyHorse video generation, Seedance 2.0 ByteDance, new open source video models 2026

### Subagent 4: GPU Self-Hosting Cost Calculator + API Services
- RunPod, Vast.ai, Lambda Labs pricing for RTX 4090, A100, H100
- fal.ai, Replicate API availability for each model
- Veo 3.1 Lite pricing ($0.05/sec) and other commercial API comparisons
- Break-even analysis

## Search Strategy
- Use wechat-article-search for Chinese-language model info (WAN, Hunyuan, SkyReels, Bernini)
- Use web_search for English-language technical details, GitHub repos, GPU benchmarks
- Time range: focus on 2025-2026 for up-to-date info

## Information Synthesis
After subagents return:
1. Compile comparison table
2. Rank models for e-commerce product-to-video
3. Calculate cost-per-video for different scales
4. Write final report with recommendations
