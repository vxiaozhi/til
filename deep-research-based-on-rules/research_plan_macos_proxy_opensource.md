Identify all major open-source proxy solutions suitable for macOS global proxy:
- Clash Verge / Clash Meta (mihomo) / Clash Nyanpasu
- V2Ray / Xray-core

- Other emerging tools

### Phase 2: Compare across dimensions

- Performance (throughput, latency, resource usage)
- GUI quality and ease of use on macOS
- Active maintenance and community
- Configuration complexity

### Phase 3: Synthesize and recommend
- Best overall recommendation
- Best for specific use cases
